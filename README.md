# My LaTex Template
This is a small Latex template.
My working stack is based on `Latex` + `VSCode` + `Git`.

## Settings
### LaTex Settings
Install a LaTex compiler.

```bash
sudo apt update
sudo apt install texlive latexmk
# Optionally if you want to write in Spanish
sudo apt install texlive-lang-spanish
```

### VSCode Settings
I encorage you to install [LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop) extension by James Yu.
